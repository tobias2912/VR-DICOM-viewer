Shaders with Intersection work with "Depth Texture", for this need turn on "Soft Particles"

Edit > Project Settings > Quality: Turn ON "Soft Particles"


"Soft Particles" require using "Deferred" or "Legacy Deferred" Rendering
http://docs.unity3d.com/Manual/RenderingPaths.html (Rendering Paths Comparison)

File > Build Settings... > Player Settings... > Other Settings: Rendering Path switch to  "Deferred" or "Legacy Deferred"
